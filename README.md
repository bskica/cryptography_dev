# Cryptography_Dev 

This repo serves as a learning implementation of various cryptography concepts.

Implemented are:
    - Caesar cipher 
    - AES (via OpenSSL)
    - Passkey generator

Planned additions:
    - PGP tools
    - Twofish (via OpenSSL and possibly from scratch) 
    - AES from scratch
    - Passkey generator with true RNG via quantum computer
